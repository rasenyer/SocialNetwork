# SocialNetwork
Social network type application developed with Kotlin and Firebase.

<h2>Used libraries:</h2>

<h4>Firebase:</h4>
• As a backend.
<br>
<h4>ImagePicker:</h4>
• To select images.
<br>
<h4>Coil:</h4>
• To set and configure the image.
